# ☕ Coffee Shop Web Application

A responsive coffee shop landing page built with HTML, CSS, and Vanilla JavaScript — inspired by real-world platforms like Starbucks.

## Features

- Dynamic menu rendering with category filtering
- Rewards system with promo code validation
- Store locator with interactive Leaflet.js map
- Real-time store search by city or area
- Feedback form with JS validation

## Tech Stack

`HTML5` &nbsp; `CSS3` &nbsp; `Vanilla JavaScript` &nbsp; `Leaflet.js`
